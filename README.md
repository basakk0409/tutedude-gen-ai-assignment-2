# Control Flow Assignment

## How to Run

control_flow_assignment.ipynb

## Tasks Covered

Task 1:
Takes user input and applies discount rules using if-elif-else.
Handles invalid input.
Calculates tax and final amount.

Task 2:
Processes a list of orders using a for loop.
Displays discount and final amount for each order.
Shows total revenue and number of discounted orders.

Task 3:
Menu-driven program using while loop.
Allows adding orders and viewing totals.
Uses break and continue.

Task 4:
Processes daily sales.
Skips zero sales using continue.
Stops on corrupted data (-1) using break.
